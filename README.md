# 🍽️ Evangadi Menu - Ethiopian Restaurant React App

## ✨ Features

- 📋 **Dynamic Menu Display** - All menu items rendered dynamically from a JavaScript data file
- 🧩 **Component-Based Architecture** - Modular, reusable React components
- 📦 **Props Drilling** - Data passed efficiently from parent to child components
- 🎨 **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- ⚡ **Fast Performance** - Built with Vite for lightning-fast development and production builds
- 🍛 **Ethiopian Cuisine Showcase** - Features 9 authentic Ethiopian dishes with descriptions and pricing

| Technology | Purpose |

| **React.js** | Frontend library for building user interfaces |
| **Class Components** | React component architecture using ES6 classes |
| **Props** | Passing data between components |
| **Vite** | Build tool and development server |
| **CSS3** | Styling and responsive design |
